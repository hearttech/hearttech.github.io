## HeartTech

## Desenvolvimento

Instalar as dependências

```
$ npm install
```

Executar uma versão hot-reload localmente

```
$ npm run start
```

## Produção

Deploy para produção. Esse comando deve fazer o build da aplicação e enviar o conteúdo da pasta `build` para a branch remota `gh-pages`.

```
$ npm run deploy
```
